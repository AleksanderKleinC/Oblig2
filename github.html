Git/Github

Git er en programvare hvor man kan se ulike versjoner av en kode. Den ser altså når endringer ble gjort, og av hvem endringene ble gjort. Man kan bruke Git til å flette/merge ulike koder sammen på en god måte. Dette er spesielt nyttig når flere jobber samtidig på et prosjekt. Git ble startet i 2005 av finsk-amerikanske Linus B. Torvalds.

Github er et nettbasert program for versjonskontroll. Dette programmet bruker Git. Github er mest brukt for kodespråk av ulike typer. Github har flere funksjonaliteter i tillegg til de som er nevnt over i Git. Noen av disse er sporing av bugs, oppgavebehandling og egne wikier for egne prosjekter.
