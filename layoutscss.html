CSS Layouts

Det finnes flere forskjellige utformingsmuligheter i CSS. Enkelte av disse er lettere å utnytte enn andre. Jeg har fire eksempler på ulike CSS layouts som jeg skal gå inn på i denne artikkelen. Disse er som følger: Flexbox, Grid, Float og Table.

CSS Flexbox

Flexbox er en endimensjonal utforming hvor du kan velge om du vil legge elementer i "columns" eller "rows". Elementene kommer til å flekse for å fylle tilgjengelig rom, eller minske for å fylle små rom. Flexbox er altså laget for å sette en rekke elementer enten nedover eller bortover. Man kan også justere mellomrom mellom fleksboksene og bestemme størrelsen på de. Flexbox og Grid blir ofte brukt om hverandre.

CSS Grid

Grid er en todimensjonal utforming. I motsetning til Flexbox kan Grid brukes både til å legge elementer nedover og bortover samtidig. Med Grid kan du bestemme utseendet på hele nettsiden. Og i likhet med Flexbox bruker også Grid "rows" og "columns" for å sette strukturen. Grid hjelper oss å kontrollere innholdet på nettstedet på en god måte. Og plasseringen av elementer blir lettere. Grid ble laget fordi Flexbox kun fungerer en retning pr. fleks.

CSS Float

Float ble i starten kun brukt til å flyte bilder inne i tekstblokker. Etterhvert ble det derimot klart at man kunne floate alle slags elementer. Float ble derfor populært for å skape kule layouts. Float er nå lite brukt til dette. Etter ankomsten til Flexbox og Grid har Float-taggen gått tilbake til orginalen. Float blir nå mest brukt for å floate bilder inne i tekst, som beskrevet over.

Les mer om Float

CSS Table

Table ble brukt før CSS var støttet ordentlig i alle nettlesere. Tabeller ble derfor brukt for å fikse nettsteders utforming og design. Man puttet på denne tiden for eksempel "header", "main" og "main" inne i tabeller for å ordne layoutet på nettstedet. Dette fungerte greit nok en stund. Men hadde mange problemer ved seg. Det var vanskelig å validere, layoutene var lite fleksible og semantisk feil. Skjermlesere hadde store problemer med å navigere inne i tabellene. I dag brukes heller Flexbox og Grid som nevnt tidligere.
